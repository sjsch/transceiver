Syntax combinator similar to many, for zero or more as, until the usable IO is exhausted.

[a a .. a]

constant :: Exp f => a -> f a -> f () Source #

Disregard the information from the covariant functor, and replace the information in the contravariaint functor with x.

For syntax combinators, this ignores the parsed token and prints the same thing every time.

padding :: Combinable f => Int -> a -> f a -> f () Source #

Syntax combinator that disregards n ts, and prints n fs.

Parsing:

_1 _2 ... _n

Printing:

f1 f2 ... f3
